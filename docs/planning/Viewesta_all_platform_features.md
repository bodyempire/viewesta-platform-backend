# Page 1

Viewesta — Full Platform Features List  
1. User Module (Viewers / Customers)  
Account & Authentication  
• Email or phone registration  
• Google / Apple / Facebook login  
• Two -step verification (optional)  
• Password reset via email  
• Profile editing (name, avatar, preferences)  
Movie Interaction  
• Browse all movies (with poster, trailer, rating, genre, description)  
• Watch movie trailers for free  
• Stream purchased movies in HD / SD quality  
• Add movies to Watchlist  
• Rate and review movies  
• Comment section (with moderation)  
• Download movies (owned a week)  
• Continue watching (resume playback from last position)  
• Subtitle and language toggle  
• Movie recommendations based on watch history  
Payments & Purchases  
• Wallet system (load money to account)  
• Payment gateway integration (MTN, Airtel, Visa, Mastercard)  
• Subscription model (monthly / yearly)  
• Pay -per-view model (one -time purchase)  
• Transaction history  
• Email and in -app receipts  
Engagement & Community  
• Like / share / comment system

2. Creator / Filmmaker Module  
Upload & Management  
• Upload movies, short films, trailers, or documentaries  
• Automatic transcoding and compression  
• Set price or mark as “Free”  
• Add subtitles / multiple languages  
• Edit movie details (title, genre, cast, tags, release date)  
Earnings & Insights  
• Dashboard showing total views, revenue, and ratings  
• Withdraw money to mobile money or bank  
• Revenue breakdown per movie  
• Creator verification badge  
• Upload analytics (viewers by region, age, device)  
3. Admin Module  
Platform Management  
• Manage all movies (approve, edit, delete)  
• Manage users and creators  
• Verify uploaded content before publishing  
• Control featured movies on homepage  
• Manage genres, tags, and categories  
Financial Management  
• Track all transactions  
• View total revenue, commission, payouts  
• Export reports (Excel/PDF)  
• Control commission rates for filmmakers

7. Technical & Backend Features  
• Secure API (JWT authentication)  
• PostgreSQL database (with AWS RDS)  
• CDN for fast video streaming  
• Cloud storage for movie files (AWS S3)  
• Email service (SendGrid / AWS SES)  
• Logging and monitoring (AWS CloudWatch)  
• Automated backups  
• Version control (GitHub)  
• CI/CD pipeline for continuous deployment  
8. Mobile App  
• Built in React Native  
• Offline downloads (stored for 7 days)  
• Push notifications  
• Dark theme only  
• When users click download  on the website, they are redirected to install/open the 
mobile app  
9. AI Recommendation System  
• Smart recommendation engine suggesting movies based on user’s viewing history 
and ratings
